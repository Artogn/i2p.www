Clean installs

  • GUI installer (requires Swing):
    install.jar (SHA1 00db8c17e26d8dfb4f654c0d7a19435ea84783c7)
    Right-click "save as", then either double click on it or run java -jar install.jar
  • Headless install:
    i2p.tar.bz2 (SHA1 6d4bc189babbe7794c18c219d22a929c9ca29dbe)
    Run (tar xjvf i2p.tar.bz2 ; cd i2p ; vi install-headless.txt)
  • Source install:
    i2p_0_4_1.tar.bz2 (SHA1 51a53152c89f9b2c7b79c70bd04b64a243ca7342)
    Alternately, you can fetch the source from cvs (CVS HEAD gets you the most recent updates, while the tag i2p_0_4_1 gets you the release)
    Run (tar xjvf i2p_0_4_1.tar.bz2 ; cd i2p_0_4_1 ; ant dist) then either run the GUI installer or headless install as above
  • After running one of the above installs, you will want to go to the router console and follow the instructions (be sure to enter your IP address / domain name, check any NAT/firewall, and reseed)

Updates from 0.4.0.1:

  1. Download i2pupdate.zip (SHA1 cd1d11554ccee6a18c7508adece4451d6b55996e) to your I2P installation directory. (alternately, you can get the source as above and run "ant updater", then copy the resulting i2pupdate.zip to your I2P installation directory)
  2. Click "Graceful restart"
  3. Wait about 2 minutes, and its done.
