Clean installs

  • GUI installer (requires Swing):
    install.jar (SHA1 d6325bd18761824bc2b6bb59855e965ce7ce412f)
    Right-click "save as", then either double click on it or run java -jar install.jar
  • Headless install:
    i2p.tar.bz2 (SHA1 d9df18fa7c10eeef7242c71b5547c9158a04353a)
    Run (tar xjvf i2p.tar.bz2 ; cd i2p ; vi install-headless.txt)
  • Source install:
    i2p_0_4_1_4.tar.bz2 (SHA1 f1318930bc5ef1114a3d367a96cb27ada690238e)
    Alternately, you can fetch the source from cvs (CVS HEAD gets you the most recent updates, while the tag i2p_0_4_1_4 gets you the release)
    Run (tar xjvf i2p_0_4_1_4.tar.bz2 ; cd i2p_0_4_1_4 ; ant dist) then either run the GUI installer or headless install as above
  • After running one of the above installs, you will want to go to the router console and follow the instructions (be sure to enter your IP address / domain name, check any NAT/firewall, and reseed)

Updates from 0.4.0.1 or later:

  1. Download i2pupdate.zip (SHA1 044f2a50d5df9b75f9067f74f6046ab3cf012fb7) to your I2P installation directory. (alternately, you can get the source as above and run "ant updater", then copy the resulting i2pupdate.zip to your I2P installation directory)
  2. Click "Graceful restart"
  3. Wait about 2 minutes, and its done.

Updates from 0.4 and earlier:

Sorry, no upgrade available. Please do a full clean install.
