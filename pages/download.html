Clean installs

  • GUI installer (requires Swing):
    i2pinstall.jar (SHA1 fadb9e606aed0b708c09191e1d4874e9be4baca4)
    Right-click "save as", then either double click on it or run java -jar i2pinstall.jar
  • Headless install:
    i2p.tar.bz2 (SHA1 9123dbd0cfd4b1b050b7a5f5c3778557f5dccf98)
    Run (tar xjvf i2p.tar.bz2 ; cd i2p ; vi install-headless.txt)
  • Source install:
    i2p-0.4.2.5.tar.bz2 (SHA1 80152291e6c92c40519bb1b55d155111140defac)
    Alternately, you can fetch the source from cvs (CVS HEAD gets you the most recent updates, while the tag i2p_0_4_2_5 gets you the release)
    Run (tar xjvf i2p-0.4.2.5.tar.bz2 ; cd i2p-0.4.2.5 ; ant dist) then either run the GUI installer or headless install as above
  • After running the GUI install on windows, simply click on the "Run I2P" button which will bring up the router console, which has further instructions.

Updates from 0.4.0.1 or later:

  1. Download i2pupdate.zip (SHA1 0af6e04266cf62538ab7c3e56b0d0c5d31ea774c) to your I2P installation directory. (alternately, you can get the source as above and run "ant updater", then copy the resulting i2pupdate.zip to your I2P installation directory)
  2. Click "Graceful restart"

Updates from 0.4 and earlier:

Sorry, no upgrade available. Please do a full clean install.
