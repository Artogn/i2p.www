Precompiled jbigi libraries (for fast native crypto)

Download the appropriate library for your operating system and CPU, save it to your I2P installation directory and restart your router.

  • Pentium2 (Linux)
  • Pentium3 (Windows) (Linux) (FreeBSD)
  • Pentium4 (Windows) (Linux) (FreeBSD)
  • i686 (Windows) (Linux) (FreeBSD)
  • Athlon (Windows) (Linux) (FreeBSD)
  • K6 (Windows) (Linux) (FreeBSD)
  • K62 (Windows) (Linux) (FreeBSD)

These are bundled under the LGPL license. Manual building instructions and signatures

Source installs

  • Fetch module 'i2p' from: :pserver:anoncvs at cvs.i2p.net/cvsroot (password: anoncvs)
  • ant installer ; java -jar ./install.jar (or guiinstall.jar)
    Note: you must have ant 1.5.*, untested on earlier versions.

Updates from 0.3.1.1-0.3.4.1

Download lib.tar.bz2 and extract it from your I2P installation directory (overwriting lib/*). In addition, if you want to try out the upcoming web console, you'll need to download weblib.tar.bz2 and webapps.tar.bz2 as well, extracting both from your I2P installation directory (adding new files to lib/ and webapps/). The contents of weblib.tar.bz2 have not changed, so if you have it already, no need to fetch it again.
