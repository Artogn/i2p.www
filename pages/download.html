Clean installs

  • GUI installer (requires Swing):
    install.jar (SHA1 9e903f7103f5e433c55d2b0f5f894f5b15895a60)
    Right-click "save as", then either double click on it or run java -jar install.jar
  • Headless install:
    i2p.tar.bz2 (SHA1 e2956d5ac6eaf2f8e3a2682c0c301deaa64a987d)
    Run (tar xjvf i2p.tar.bz2 ; cd i2p ; vi install-headless.txt)
  • Source install:
    i2p_0_4_1_3.tar.bz2 (SHA1 24e5a3ec8f3efa8b52d4f42c78de72697ed35105)
    Alternately, you can fetch the source from cvs (CVS HEAD gets you the most recent updates, while the tag i2p_0_4_1_3 gets you the release)
    Run (tar xjvf i2p_0_4_1_3.tar.bz2 ; cd i2p_0_4_1_3 ; ant dist) then either run the GUI installer or headless install as above
  • After running one of the above installs, you will want to go to the router console and follow the instructions (be sure to enter your IP address / domain name, check any NAT/firewall, and reseed)

Updates from 0.4.0.1 or later:

  1. Download i2pupdate.zip (SHA1 da875130f7dcbaaf055716a6c62818465ce4302f) to your I2P installation directory. (alternately, you can get the source as above and run "ant updater", then copy the resulting i2pupdate.zip to your I2P installation directory)
  2. Click "Graceful restart"
  3. Wait about 2 minutes, and its done.

Updates from 0.4 and earlier:

Sorry, no upgrade available. Please do a full clean install.
