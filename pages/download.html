Clean installs

  • GUI installer (requires Swing):
    install.jar (SHA1 63ef5a4e1f499db0b269c72959719c78c85902dc)
    Right-click "save as", then either double click on it or run java -jar install.jar
  • Headless install:
    i2p.tar.bz2 (SHA1 f77a7021fa5e18e9a7125e5dd8bb83e023efa435)
    Run (tar xjvf i2p.tar.bz2 ; cd i2p ; vi install-headless.txt)
  • Source install:
    i2p-0.4.2.3.tar.bz2 (SHA1 f505afcc7f8f34e771cc7de6bb9cf1dd1058bb44)
    Alternately, you can fetch the source from cvs (CVS HEAD gets you the most recent updates, while the tag i2p_0_4_2_3 gets you the release)
    Run (tar xjvf i2p-0.4.2.3.tar.bz2 ; cd i2p-0.4.2.3 ; ant dist) then either run the GUI installer or headless install as above
  • After running the GUI install on windows, simply click on the "Run I2P" button which will bring up the router console, which has further instructions.

Updates from 0.4.0.1 or later:

  1. Download i2pupdate.zip (SHA1 4edc182741be38752a050d1cbfeaeb3ad4e52e7c) to your I2P installation directory. (alternately, you can get the source as above and run "ant updater", then copy the resulting i2pupdate.zip to your I2P installation directory)
  2. Click "Graceful restart"

Updates from 0.4 and earlier:

Sorry, no upgrade available. Please do a full clean install.
