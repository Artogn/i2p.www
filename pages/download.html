Clean installs

  • GUI installer (requires Swing):
    install.jar (SHA1 5a92ffdac4edce942faa2f8fa3b9c468f646a6db)
    Right-click "save as", then either double click on it or run java -jar install.jar
  • Headless install:
    i2p.tar.bz2 (SHA1 455b936f0b49ee58ab50739e7b00a482678b9291)
    Run (tar xjvf i2p.tar.bz2 ; cd i2p ; vi install-headless.txt)
  • Source install:
    i2p_0_4_1_2.tar.bz2 (SHA1 124ce2e680f8a194d573edc9e688c6ab1f085d05)
    Alternately, you can fetch the source from cvs (CVS HEAD gets you the most recent updates, while the tag i2p_0_4_1 gets you the release)
    Run (tar xjvf i2p_0_4_1_2.tar.bz2 ; cd i2p_0_4_1_2 ; ant dist) then either run the GUI installer or headless install as above
  • After running one of the above installs, you will want to go to the router console and follow the instructions (be sure to enter your IP address / domain name, check any NAT/firewall, and reseed)

Updates from 0.4.0.1 or later:

  1. Download i2pupdate.zip (SHA1 e6f140f9a4ccdb59e3784510c9bff5d336dafca4) to your I2P installation directory. (alternately, you can get the source as above and run "ant updater", then copy the resulting i2pupdate.zip to your I2P installation directory)
  2. Click "Graceful restart"
  3. Wait about 2 minutes, and its done.

Updates from 0.4 and earlier:

Sorry, no upgrade available. Please do a full clean install.
