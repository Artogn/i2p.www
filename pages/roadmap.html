0.5 (January 2005)

  • Update the tunnel pools to allow per-dest outbound tunnels, smaller free pools, more aggressive exploration, strict ordering, and randomization of tunnel length
  • Revised tunnel IDs and encryption to deal with colluding attackers within a tunnel
  • Address load caused by i2p-bt by throttling data in tunnels as well as participation in tunnels
  • Tech docs

0.6 (April 2005)

  • UDP transport (including support for some NATs/firewalls)
  • Basic content distribution infrastructure, suitable for address books, blogs, and forums
  • html/http filter

1.0 (May 2005)

  • Working: blogs, chat, mail, news, BT, eepsites, search
  • User docs

1.1 (June 2005)

  • NetworkDB improvements
  • (S)NTP improvements
  • Streaming lib optimization

2.0 (September 2005)

  • Full restricted routes

3.0 (January 2006)

  • Tunnel mixing and padding
  • User defined message delays
