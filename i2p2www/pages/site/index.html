{% trans %}What is I2P?{% endtrans %}

  • {% trans -%} I2P is a full darknet implementation - a network within a network, unmonitorable by your ISP or government. {% endtrans %}
  • {% trans -%} I2P is used by many people who care about their privacy, as well as those in high-risk situations such as activists, oppressed people, journalists and whistleblowers. {% endtrans %}
  • {% trans -%} I2P runs on Java and is therefore available anywhere Java will run, including desktops, embedded systems (like the Raspberry Pi) and Android phones. {% endtrans %}
