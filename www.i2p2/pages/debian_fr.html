Paquets I2P pour Debian

The paquets hébergés sur le site Launchpad I2P ont été testés et devraient fonctionner sur x86/x86_64 avec
  • Ubuntu (Hardy 8.04 et ultérieures)
  • Mint 11
  • Debian Linux (Lenny et ultérieures) & kFreeBSD (Wheezy)
  • Knoppix
  • Simply MEPIS 8.5.03-rel1
  • Trisquel 4.5.1
  • gNewSense 2.3
  • Nexenta 3.0.1
Les paquets I2P peuvent marcher sur des systèmes non listés ci-dessus. Merci de rapporter tout problème avec ces paquets dans Trac sur http://trac.i2p2.de.
  • Option 1: Versions récentes d'Ubuntu et ses dérivées (À essayer si vous n'utilisez pas Debian)
  • Option 2: Debian (dont les systèmes basés sur Debian et les anciennes versions d'Ubuntu)
  • Option 3: Non-Linux (et peut-être les architectures non x-86)

Instructions pour Ubuntu Lucid Lynx (et plus récentes) et ses dérivées comme Linux Mint & Trisquel

Ajout du PPA via la ligne de commande et installation d'I2P
  1. Ouvrez un terminal et entrez:
        sudo apt-add-repository ppa:i2p-maintainers/i2p
    Cette commande ajoutera le PPA à /etc/apt/sources.list.d et répliquera la clé gpg avec laquelle le dépôt à été signé. La clé GPG assure que le paquet n'a pas été modifié depuis sa fabrication.
  2. Indiquez le nouveau PPA à votre gestionnaire de paquets
        sudo apt-get update
    Cette commande va récupérer la dernière liste de logiciels de chaque dépôt activé sur votre système, donc le PPA I2P ajouté par la commande précédente.
  3. Vous êtes maintenant prêt à installer I2P!
        sudo apt-get install i2p
Ajout du PPA et installation avec Synaptic
  1. Ouvrez Synaptic (Système -> Administration -> Gestionnaire de paquets Synaptic).
  2. Une fois Synaptic lancé, sélectionnez Dépôts dans le menu Configuration.
  3. Cliquez sur l'onglet Autres logiciels et cliquez sur le bouton Ajouter.... Collez ppa:i2p-maintainers/i2p dans le champ "Ligne APT puis cliquez sur le bouton + Ajoutez une source de mise à jour. Cliquez sur le bouton Fermer (et éventuellement une deuxième fois sur la notification de modification de dépôts "Dépôts modifiés", puis sur le bouton du menu Recharger.
  4. Dans le champ "Filtre rapide", tapez i2p. Quand i2p s'affiche dans la liste de résultats, faites dessus un clic-droit et choisissez Sélectionner pour installation. Après ceci, vous verrez peut-être une fenêtre Prévoir d'effectuer d'autres changements?. Si oui, cliquez sur Ajouter à la sélection puis dans le menu sur le bouton Appliquer.
À la fin du processus d'installation vous pouvez passer à l'étape suivante pour démarrer I2P et le configurer pour votre système.

Instructions pour Debian Lenny et plus récentes

Les étapes suivantes doivent être effectuées avec l'accès root (c.à d. en basculant de l'utilisateur en cours à root avec "su" ou en préfixant chaque commande avec "sudo").
  1. Ajoutez les entrées suivantes à /etc/apt/sources.list.d/i2p.list
        deb http://ppa.launchpad.net/i2p-maintainers/i2p/ubuntu natty main
        deb-src http://ppa.launchpad.net/i2p-maintainers/i2p/ubuntu natty main
    Ces deux lignes devraient fonctionner quelle que soit la version de Debian installée.
  2. Ajouter la clé GPG de signature du dépôt avec la commande suivante:
        apt-key adv --keyserver keyserver.ubuntu.com --recv-keys EB2CC88B
    Vous obtiendrez une sortie semblable à la suivante si la commande a réussi:
        
  3. Instruisez votre gestionnaire de paquets du nouveau PPA en entrant
        apt-get update
    Cette commande va récupérer la dernière liste de logiciels depuis chaque dépôt activé sur votre système, dont ceux du PPA I2P ajouté à l'étape 1.
  4. Vous êtes maintenant prêt à installer I2P!
        apt-get install i2p
À la fin du processus d'installation vous pouvez passer à l'étape suivante pour démarrer I2P et le configurer pour votre système.

Instructions pour Non-Linux / Non-x86

Les étapes suivantes doivent être effectuées avec l'accès root (c.à d. en basculant de l'utilisateur en cours à root avec "su" ou en préfixant chaque commande avec "sudo").
  1. Téléchargez le paquet i2p-router depuis le PPA.
  2. Assurez-vous que Java est installé. Lancer apt-get install default-jvm devrait suffire.
    Vous êtes maintenant prêt à installer I2P!
        dpkg -i i2p-router_*.deb
  3. On recommande de compiler jbigi pour votre système pour obtenir les meilleures performances. Les instructions sont disponibles ici.
À la fin du processus d'installation vous pouvez passer à l'étape suivante pour démarrer I2P et le configurer pour votre système.

Après l'installation

L'utilisation de ces paquets I2P vous permet de lancer le routeur de l'une de ces trois façons:
  • "À la demande", en utilisant le script i2prouter. Lancez simplement "i2prouter start" depuis une invite de commande. (Note: N'utilisez pas sudo, ni ne l'exécutez en tant que root!)
  • "À la demande" sans le wrapper de service java (requis sur les systèmes non-Linux/non-x86) en lançant "i2prouter-nowrapper". (Note: N'utilisez pas sudo, ni ne l'exécutez en tant que root!)
  • "En tant que service" qui démarre en même temps que le système, avant même l'ouverture de session. Le service peut être activé avec "dpkg-reconfigure i2p" en tant que root ou avec sudo. C'est la méthode recommandée.

À la première installation, pensez à régler votre pare-feu/NAT si vous pouvez, en gardant à l'esprit les ports tournés vers Internet qu'utilise I2P, décrits ici entre autres. Si vous avez correctement ouvert votre port TCP aux connexions entrantes, activez également les connexions TCP entrantes sur la page de configuration.

Pour finir, contrôlez et réglez votre bande passante sur la page de configuration, les réglages par défaut de 96 ko/s descendants / 40 ko/s montant étant volontairement trop faibles pour une utilisation normale.

Si vous voulez atteindre des sites eep avec votre navigateur, consultez la page de réglage du proxy de navigateur pour une méthode facile.

Vous pourriez préférer utiliser I2Pfox, un profil prêt à l'emploi. {% endblock %}
